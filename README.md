# Ledgerly

Ledgerly is a full-stack analytics SaaS for bar and restaurant management that tracks sales, ingredients, recipes, menus, and financial insights through a modern dashboard.

## Tech Stack

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS

### Backend

- Node.js
- Express
- TypeScript

### Planned

- Prisma
- PostgreSQL
- Docker
- GitHub Actions
- Analytics dashboard
- Auth and role-based access
